=========================
Using as a Python library
=========================

csvkit is designed to be used a replacement for most of Python's :mod:`csv` module. Important parts of the API are documented on the following pages.

Don't!

.. code-block:: python

    import csv

Do!

.. code-block:: python

    import csvkit
